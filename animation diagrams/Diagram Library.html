integration_instructions How to insert a block in a slide
1. Load the plugin file in your slide deck
<!-- inside <head> -->
<script type="text/babel"
        src="route-diagram.jsx"></script>
2. Drop the block where you want it
<RouteDiagram
  stops={[...]}
  speed={1}
  showBattery
/>
Each plugin lives as <name>-diagram.jsx next to your slide HTML. Open Route studio arrow_forward
